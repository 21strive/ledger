# Migrations Directory

This directory will contain the versioned migration files generated by Atlas.

Migration files will be automatically created when you run:

```bash
atlas migrate diff migration_name \
  --env local \
  --dir file://database/migrations \
  --to file://database/schemas/schema.sql
```

## Migration File Format

Atlas generates migration files in the format:

- `YYYYMMDDHHMMSS_migration_name.sql` - The migration SQL
- `atlas.sum` - Checksum file for migration integrity

## Initial Migration

To create your first migration from the current schemas:

```bash
cd /Users/mrifqi/personal/gitrepo/ledger

atlas migrate diff initial_schema \
  --env local \
  --dir file://database/migrations \
  --to file://database/schemas/schema.sql
```

This will generate the initial migration based on all the schema files.
